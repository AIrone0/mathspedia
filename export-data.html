Export Mathspedia Data

This page will export all data from your browser's localStorage to a JSON file that can be imported into MediaWiki.

Instructions:
  1. Click "Export Data" below
  2. A JSON file will be downloaded
  3. Save it as data-export.json in the project root
  4. Run: php migrate-to-mediawiki.php data-export.json
